# 🐰 Sorry Bunny - Maafi Chahiye!

Ek pyara sa animated bunny jo maafi maangta hai! "Maaf Kiya" button dabao aur dekho kya hota hai 💕

## Features
- 🐰 Cute animated bunny (CSS se bana, koi image nahi!)
- 😢 Sad bunny with tears jo maafi maangta hai
- 🎉 Confetti blast jab maafi mil jaaye
- 💬 "Nahi" button har baar funny response deta hai
- 📱 Mobile friendly

## Local Chalane ka Tareeqa

```bash
npm install
npm run dev
```

Phir browser mein jaao: `http://localhost:5173`

## 🚀 GitHub Pe Upload Kaise Karein

### Step 1 — GitHub pe naya repo banao
1. [github.com](https://github.com) pe jao
2. **New repository** click karo
3. Naam likho: `sorry-bunny`
4. **Create repository** dabao

### Step 2 — Terminal mein yeh commands chalao

```bash
# Project folder mein jao
cd sorry-bunny

# Git shuru karo
git init

# Sab files add karo
git add .

# Pehla commit karo
git commit -m "🐰 Sorry bunny - initial commit"

# GitHub repo connect karo (apna username daalo)
git remote add origin https://github.com/TUMHARA_USERNAME/sorry-bunny.git

# Upload karo!
git push -u origin main
```

### Step 3 — Free hosting (optional) — Vercel ya Netlify

**Vercel se:**
```bash
npm install -g vercel
vercel
```

**Ya** [vercel.com](https://vercel.com) pe jao → Import GitHub repo → Deploy! ✨
